Company Run：POP MART / PPMT

  • Date: 2026-07-08
  • Ticker / market: 9992.HK / Hong Kong
  • Depth: Smoke
  • Core question: Labubu / THE MONSTERS 的 2025 爆发，到底证明了全球 IP 平台能力，还是把利润推到了一个需要正常化的高点？

结论先说：Pop Mart 不是现金差、财务假、马上要出事的公司。它真正难判断的地方是：2025 年的收入、利润率、海外扩张和估值，很多都压在同一个变量上——THE MONSTERS / Labubu 的热度能不能沉淀成长期 IP 资产。

1. Business Snapshot

1.0 Value Line-style Tearsheet

  • 公司一句话：

    • Pop Mart 是一家 IP 消费品公司。它把艺术家角色、产品设计、稀缺发售、直营渠道和粉丝社群放在一起，卖潮玩、毛绒、手办和衍生品。
    • 2025 年以后，投资问题已经不只是“中国潮玩零售还能不能增长”，而是“Labubu 爆发后，公司能不能证明自己是全球 IP 平台”。

  • 谁付钱 / 为什么买：

    • 主要消费者是年轻人、收藏者、粉丝和礼品购买者。购买理由不是功能，而是情绪价值、审美、身份表达、收藏和社交。
    • 招股书把 pop toys 定义成面向年轻成人的文化消费。这一点很重要，因为它解释了高毛利，也解释了为什么热度、稀缺和社媒传播会同时影响需求。

  • 收入来源：

    • 2025 收入 RMB 37.12bn，其中自主产品占 99.1%，Artist IP 占 90.0%。
    • THE MONSTERS 一项收入 RMB 14.16bn，占总收入 38.1%。中国收入 RMB 20.85bn，海外收入 RMB 16.27bn。
    • 所以 Pop Mart 已经不是单纯中国门店生意。它的边际变化来自海外、线上、THE MONSTERS 和毛绒品类。

  • 利润来源：

    • 利润来自高毛利 IP 商品、直营渠道和 2025 年快速放大的经营杠杆。
    • 2025 毛利率到 72.1%，净利率到 35.1%。这说明公司变现能力很强，但也说明估值不能直接把 2025 当成常态。

  • 财务强度：

    • 2025 年底现金及等价物 RMB 13.78bn，公司披露净现金 RMB 10.92bn。短期不是融资生存问题。
    • 但资产负债表变重：存货从 2023 年 RMB 0.91bn 到 2025 年 RMB 5.47bn，租赁负债从 RMB 0.78bn 到 RMB 2.86bn。热度继续强时，这是增长弹药；热度回落时，它会放大波动。

  • 管理层 / 文化：

    • 王宁仍任董事长兼 CEO。招股书强调 speed、teamwork、creativity，适合新品、潮流和快速试错。
    • 2025 年报继续强调长期孵化 IP 和全球化组织调整。但顺风期讲长期主义不难，真正要看的，是热度下降时管理层会不会控制补货、放慢开店、保护品牌，而不是继续用新故事覆盖旧问题。

  • 3-5 年决定价值的变量：

    • THE MONSTERS 能否从爆款变成 evergreen IP。
    • 非 THE MONSTERS 的 IP 能否接力，而不是只在年报里证明“还有很多 IP”。
    • 海外成熟店是否有稳定同店和 payback，而不是只吃新店、排队和 TikTok 热度。
    • 毛绒品类占比下降后，毛利率还能留住多少。
    • 现金是否投向高 ROIC 扩张，而不是追热度开店、做泛娱乐项目或高价回购。

  • 一句话反方：

    • 2025 年的 Labubu / THE MONSTERS 超级周期同时推高收入、毛利率、渠道扩张和估值。只要热度退潮，这些变量可能不是一个一个回落，而是一起回落。

1.1 一句话

Pop Mart 是一家财务质量很强、执行能力很强、但正处在高点验证期的 IP 消费公司。它不是典型弱公司；风险也不在短期现金流。风险在于投资人可能把 2025 年由爆款、稀缺、毛绒 mix 和海外早期红利共同推高的利润，当成了长期平台利润。

1.2D 衍生指标怎么服务 checklist

  • 质量指标：
    • 毛利率、经营利润率、净利率用于判断 2025 是否是利润率高点。
    • ROE 用粗略期末权益口径看资本回报，但需要 Full run 再用平均权益、ROIC 和投入资本口径重算。
  • 现金指标：
    • 经营现金流 / 净利润、自由现金流代理 / 净利润用于判断利润有没有变成可分配现金。
    • 资本开支 / 收入用于判断增长是否越来越依赖重投入。
  • 周期和承诺指标：
    • 存货 / 收入、租赁负债 / 收入用于判断热度是否被固化成资产负债表承诺。
    • 这些指标比单看收入更适合触发 Q12 和 Q17，因为它们会先显示“需求降速后，成本和库存会不会跟着降”。

1.3 Business Structure

  • 按 IP 看，平台能力和集中度风险同时存在。

    • 支持平台能力的证据：2025 年 17 个 Artist IP 收入超过 RMB 100m，6 个 IP 超过 RMB 2bn。
    • 支持集中度风险的证据：THE MONSTERS 一项就贡献 RMB 14.16bn，占总收入 38.1%，同比 +365.7%。
    • 所以问题不是“Pop Mart 有没有其他 IP”。真正问题是：当 THE MONSTERS 降速时，其他 IP 能不能承担增量，而不是只承担存量。

  • 按品类看，2025 的利润和毛绒高度相关。

    • 毛绒收入 RMB 18.71bn，占总收入 50.4%，同比 +560.6%。
    • 手办收入 RMB 12.02bn，占 32.4%，同比 +73.3%。
    • 这说明 2025 不是所有品类均匀变强，而是毛绒成为核心增量。毛绒如果回到正常增速，利润率也要重新估。

  • 按地区看，海外已经是第二条增长曲线，但还没有证明成熟期经济。

    • 中国收入 RMB 20.85bn，海外收入 RMB 16.27bn。
    • 美洲 RMB 6.81bn，同比 +748.4%；欧洲及其他 RMB 1.45bn，同比 +506.3%。
    • 这些数字证明海外 demand capture 很强，但不能直接证明成熟店同店、复购和 payback。

  • 按渠道看，线上能力强，也更容易受 viral demand 影响。

    • 美洲线上收入 RMB 4.35bn，占美洲 64.0%；其中 TikTok 美洲收入 RMB 1.04bn。
    • 这说明公司会抓流量，也说明增长对平台流量、短视频传播和爆款情绪更敏感。

1.4 Business System

  • Pop Mart 的利润池来自 IP 情绪价值，不来自制造壁垒。

    • 高毛利的来源是角色、设计、稀缺和粉丝关系。
    • 制造端本身不是最深护城河；最深的部分是能不能反复把角色做成产品、社群和长期购买理由。

  • 公司必须同时做两件互相拉扯的事。

    • 一边要制造稀缺，维持角色热度和收藏欲。
    • 另一边要补货、开店、扩品类，承接全球需求。
    • 如果补得太慢，收入被黄牛和二级市场吃掉；如果补得太快，稀缺感和二级价格可能先掉，库存和折扣再跟上。

  • 因此，2026 以后最重要的不是收入还增不增长，而是增长从哪里来。

    • 来自非 THE MONSTERS、成熟店、复购和多品类延展，说明平台能力在接棒。
    • 来自继续缺货、社媒炒作、新店堆量和二级价支撑，说明 2025 周期仍在拉长，但质量没有完全证明。

1.5 Core Capabilities vs Counterevidence

  • IP 孵化和商业化：

    • 支持证据：多个十亿级 IP 已经存在，THE MONSTERS 也不是凭空出现，Pop Mart 确实有把艺术家角色商品化的系统。
    • 反证/疑点：2025 的增量太集中在 THE MONSTERS 和毛绒。平台能力不能只用“有很多 IP”证明，必须用“爆款降速后还有谁接力”证明。

  • 全球化复制：

    • 支持证据：2025 海外收入 RMB 16.27bn，美洲和欧洲增速极高。
    • 反证/疑点：2026 Q1 海外仍增长但明显降速。没有成熟店同店、区域利润率和 payback 时，不能把早期海外收入直接当成长期门店经济。

  • 渠道运营：

    • 支持证据：公司能同时跑直营门店、roboshop、官网/APP、TikTok 等渠道。
    • 反证/疑点：线上高增可能包含 viral demand。渠道抓住热度不等于渠道本身能创造长期需求。

  • 财务变现：

    • 支持证据：2025 毛利率、净利率、现金余额和经营现金流都很强。
    • 反证/疑点：存货、租赁负债和人员快速上升。财务强不等于没有周期，只说明它有更长时间去证明自己。

1.5B Management / Culture Evidence

  • 王宁和管理层的优势，是很懂 IP、产品和渠道速度。

    • 招股书把组织文化写成 speed、teamwork、creativity，这和潮流消费品的成功机制一致。
    • 2025 年报里，王宁把 THE MONSTERS 描述为“金矿”，并认为价值挖掘刚开始。这说明管理层对 IP 生命周期仍然乐观。

  • 投资上要看的反面，是顺风期的乐观会不会变成高点外推。

    • 高热度时继续开店、补货、扩品类，看起来都合理。
    • 但如果热度下降，真正优秀的管理层要能主动放慢，把库存、折扣、门店和品牌稀缺性放在收入增速前面。

  • 这块还缺系统证据。

    • 需要补 2019-2026 王宁和核心管理层访谈时间线。
    • 需要看激励是否绑定 ROIC、现金流和每股价值，而不是收入、GMV 或声量。
    • 还需要观察逆风动作：降产、少开店、削新品、保护品牌，还是继续讲新叙事。

1.6 Strong Bear Cases + Known/Unknown

  • Bear case 1：Labubu 高点。

    • 为什么可能是真的：THE MONSTERS 占 38.1%，毛绒占 50.4%，二者都和 2025 热度高度相关。
    • 这会同时影响收入、毛利率、库存、海外渠道和估值，所以它不是普通单品风险。
    • 需要什么证据推翻：2026 分 IP sell-through、二级价、库存 aging、非 THE MONSTERS 新品增长。

  • Bear case 2：海外扩张追高。

    • 为什么可能是真的：美洲和欧洲门店、租赁负债、销售人员都快速增加；2026 Q1 海外增长明显低于 2025 全年。
    • 这不代表海外失败，但说明海外已经从“爆发”进入“验证”。
    • 需要什么证据推翻：成熟店同店、单店 payback、区域利润率、关店成本。

  • Bear case 3：管理层把高热度当成长期平台能力。

    • 为什么可能是真的：2025 年报语言乐观，2026 年初仍有回购和多线扩张。
    • 这不一定错，但必须由后续动作证明：管理层是否愿意牺牲短期收入，保护 IP 生命周期和渠道质量。
    • 需要什么证据推翻：热度下降时主动控货、收缩低效扩张、提高披露透明度。

2. Checklist Triggers → Evidence Chase

2.0 这不是五个风险点，而是一条因果链

这章重写成一条链，而不是 checklist 项目的堆叠。Pop Mart 当前最重要的失败路径是：

THE MONSTERS / 毛绒爆发
→ 2025 收入和利润率被推到高位
→ 管理层把高需求转成海外门店、库存、租约和人员
→ 投资者需要验证现金是否能回到股东、扩张是否有真实 ROI
→ 如果披露跟不上复杂度，市场会更难区分平台能力和周期红利

这条链里，Q19 是起点，Q17 是利润表结果，Q12 是资产负债表结果，Q21 是股东可得性检查，Q15 是披露和资本配置检查。后面的 trigger 不是和前面并列，而是在问同一个问题的下一层：如果 2025 的需求不是长期常态，那么哪些数字会先被重估。

Trigger：Q19 脆弱集中度 / 单一 IP 和单一品类反转

  • Severity: High
  • 当前状态：Watch
  • Failure Mechanism: THE MONSTERS 和毛绒如果只是一个超级周期，收入不会单独回落；毛利率、库存、海外开店节奏和估值倍数会一起受压。
  • Cohesive judgment: 这个 trigger 是整条链的起点。Pop Mart 不是单 IP 公司，因为它确实有多个十亿级 IP；但 2025 年边际增长太集中在 THE MONSTERS 和毛绒上。投资上真正要问的不是“有没有其他 IP”，而是“当 THE MONSTERS 降速时，其他 IP 能否接住增量和毛利率”。
  • Evidence Chain:
    • 事实：2025 THE MONSTERS 收入 RMB 14.16bn，占总收入 38.1%，同比 +365.7%；毛绒收入 RMB 18.71bn，占 50.4%，同比 +560.6%。
    • 推论：这两个维度表面不同，底层却可能是同一个需求变量：Labubu 热度把 IP 和品类一起推高。
    • 反证：公司还有 6 个 IP 超过 RMB 2bn、17 个 Artist IP 超过 RMB 100m，因此不能把 Pop Mart 简化成“只有 Labubu”。
    • 缺口：还没有 2026 分 IP sell-through、二级价序列、库存 aging 和非 THE MONSTERS 新品复购数据。
  • What would change the view: 如果 2026-2027 非 THE MONSTERS 继续增长，且 THE MONSTERS 二级价回落后 一手 sell-through 仍强，这个 trigger 会降权。相反，如果非 THE MONSTERS 只能维持存量、毛绒占比下降后毛利率同步回落，这个 trigger 会从 Watch 接近 Fail。
  • 是否需要用户判断：是

Trigger：Q17 周期正常化 / 是否把高点利润当正常利润

  • Severity: High
  • 当前状态：Watch
  • Failure Mechanism: 爆款周期会让收入、毛利率、经营杠杆和市场叙事同时变好；一旦投资人把这组数字当成正常利润，估值会系统性偏高。
  • Cohesive judgment: Q17 是 Q19 在利润表上的结果。2025 利润是真实的，现金流也很强；问题是这组利润是否可重复。THE MONSTERS 和毛绒把收入结构变了，海外和线上把渠道结构变了，经营杠杆把净利率推高了。只要这些变量有一个正常化，2025 的净利率就不能直接外推。
  • Evidence Chain:
    • 事实：2025 收入 YoY +184.7%，经营利润 YoY +306.6%，毛利率 72.1%，净利率 35.1%。2026 Q1 总收入仍同比 +75%-80%，但已经低于 2025 全年爆发速度。
    • 推论：2025 不是会计幻觉，但它更像高点验证年，而不是已经证明长期平台利润率。
    • 反证：Q1 2026 仍高增长，中国区同比 +100%-105%，说明需求没有立刻消失。
    • 缺口：缺 2026 H1 毛利率、分品类毛利率、库存跌价、折扣率、退货率和海外同店。
  • What would change the view: 如果 2026-2027 毛利率在库存更高、热度更正常的情况下仍接近 2025，且经营现金流持续覆盖净利润，这个 trigger 会降权。否则，估值要用 normalized margin，而不是 2025 margin。
  • 是否需要用户判断：是

Trigger：Q12 固定承诺错配 / 海外扩张是否锁定了高点需求

  • Severity: Medium
  • 当前状态：Watch
  • Failure Mechanism: 当公司把高热度需求转成门店、租约、人员和库存后，需求如果降速，原本可变的消费热度会变成固定或半固定成本压力。
  • Cohesive judgment: Q12 是 Q19 和 Q17 在资产负债表上的结果。Pop Mart 现金很多，所以这不是流动性危机；真正的问题是管理层是否把 2025 的需求信号当成长期需求，提前锁定了 2026-2028 的门店、租约和库存。
  • Evidence Chain:
    • 事实：2025 租赁负债 RMB 2.86bn，2024 为 RMB 0.96bn；存货 2025 RMB 5.47bn，YoY +258.9%。美洲门店从 22 间到 64 间，欧洲及其他地区门店从 14 间到 36 间。
    • 推论：如果海外成熟店有稳定复购，这些投入会变成获客资产；如果 2025 主要是排队、缺货和 TikTok 热度，这些投入会放大利润下行。
    • 反证：公司净现金 RMB 10.92bn，且 2026 Q1 海外仍增长，美洲 +55%-60%、欧洲及其他 +60%-65%。
    • 缺口：缺海外成熟店同店、单店 payback、租约期限、退出成本、分地区库存 aging 和区域利润率。
  • What would change the view: 如果成熟店同店继续增长、新店 payback 没有拉长、库存不靠折扣消化，Q12 降权。相反，如果海外收入靠新店堆量但同店弱、库存 aging 上升，Q12 会成为 Q19 之后最重要的确认信号。
  • 是否需要用户判断：是

Trigger：Q21 表外/关联方/跨境结构可验证性

  • Severity: Low
  • 当前状态：Clear
  • Failure Mechanism: 跨境结构只有在现金不能回到上市公司、关联交易转移价值、或表外安排隐藏风险时，才会伤害普通股东。
  • Cohesive judgment: Q21 不是这条链的主要风险，而是股东可得性检查。前面三个 trigger 都在问“利润是否可持续”；Q21 问的是“就算利润存在，普通股东能不能拿到”。目前公开资料没有显示 Enron/FTX 式隐藏负债或客户资产转移，但 Full run 仍应做现金归集和关联交易穿透。
  • Evidence Chain:
    • 事实：公司是 Cayman 上市主体，2025 年中国内地子公司未分配利润 RMB 20.16bn；年报披露递延税处理，2026 年初也有回购。
    • 推论：结构本身不是问题，现金路径和股东回报路径才是问题。
    • 反证：审计意见正常，当前没有看到严重关联交易、SPE 隐藏亏损或客户资产转移证据。
    • 缺口：缺子公司现金分布、分红税务路径、关联交易穿透和法律结构专项。
  • What would change the view: 如果公司持续分红或回购，且现金能稳定从经营实体回到上市主体，Q21 继续维持低权重。若出现大额关联交易、现金困在子公司或不透明资金流，Q21 才会升权。
  • 是否需要用户判断：否

Trigger：Q15 复杂度 / 投资者是否能拆清真实利润来源和资本配置

  • Severity: Medium
  • 当前状态：Watch
  • Failure Mechanism: 公司从清晰的 IP 商品生意扩到电影、主题乐园、甜品、授权和跨界项目后，投资者可能看不清每一块业务的 ROI；复杂度本身不是风险，ROI 被遮住才是风险。
  • Cohesive judgment: Q15 是这条链的最后一环。当前利润主线仍然清楚：THE MONSTERS、毛绒、海外渠道。但如果管理层用更多新业务承接 2025 热度，披露却仍停留在大类收入和叙事层面，投资者会更难判断现金是在强化核心 IP，还是流向低回报扩张。
  • Evidence Chain:
    • 事实：2025 年报提到毛绒、饰品、甜品、电影等新业务探索，公司也在多地区、多渠道同步扩张。
    • 推论：新业务可能增强 IP 生命周期，也可能稀释资本纪律；判断关键是分项目 ROI 和关停机制。
    • 反证：目前主题乐园、授权、甜品等还不是利润主线，公司现金也足，短期没有靠复杂会计掩盖经营问题。
    • 缺口：缺新业务投入额、亏损、收入、回收期，以及管理层如何关停失败项目。
  • What would change the view: 如果新业务投入小、试错快、披露变细，并且核心 IP 商品现金流没有被消耗，Q15 降权。若新业务不断扩大但 ROI 不披露，Q15 会放大 Q12 和 Q17 的不确定性。
  • 是否需要用户判断：是

2.1 当前最短判断

这条因果链目前还没有走到 Fail，因为公司现金强、Q1 2026 仍高增长、IP 矩阵也不薄。它也不能被判 Clear，因为 2025 的利润率、库存、租赁负债和海外开店都被同一个热度变量推着走。下一次研究不用再加更多普通风险，应该只追三个断点：非 THE MONSTERS 能否接力，2025 利润率能否正常化后仍高，海外成熟店能否证明 payback。

3. Next Evidence / User Judgment

3.1 当前判断

Pop Mart 的投资判断现在不能靠一句“Labubu 泡沫”或“全球 IP 平台”解决。公开证据支持一个中间判断：公司质量很高，2025 利润也有现金支撑；但最强增长变量、最高利润率变量和最快扩张变量，都和 THE MONSTERS / 毛绒 / 海外热度有关。因此，下一步不是继续加普通风险，而是把 2025 利润拆成可持续部分和周期部分。

3.2 下一步证据追查

  • P0：2026 分 IP sell-through。

    • 为什么重要：它直接回答 THE MONSTERS 是否降温，以及其他 IP 是否接力。
    • 当前状态：Searching
    • 下一步：等 H1 2026 或找渠道数据。

  • P0：库存 aging、折扣率、退货/滞销。

    • 为什么重要：爆款退潮通常先体现在库存和折扣，而不是利润表。
    • 当前状态：Blocked
    • 为什么无法继续：年报只披露存货余额和减值，没有披露 aging 和渠道折扣。

  • P0：海外成熟店 cohort、同店、区域利润率。

    • 为什么重要：它区分“全球复制能力”和“追高开店”。
    • 当前状态：Blocked
    • 为什么无法继续：公开年报没有成熟店 cohort，需要业绩会、渠道调研或第三方数据。

  • P0：二级市场价格和社媒热度序列。

    • 为什么重要：它能帮助判断 Labubu 热度是从稀缺转成长期需求，还是从稀缺转成疲劳。
    • 当前状态：Searching
    • 下一步：抓 StockX/eBay/闲鱼、Google Trends、TikTok/小红书。

  • P1：王宁访谈和逆风行为。

    • 为什么重要：顺风讲长期主义不难，难的是热度降温时保护品牌和资本回报。
    • 当前状态：Searching
    • 下一步：整理 2019-2026 访谈时间线。

3.3 需要用户判断的问题

  • THE MONSTERS 到底是什么？

    • 判断 A：evergreen IP。
      • 可以接受更高 normalized margin。
    • 判断 B：2025 社媒/稀缺周期。
      • 必须大幅下修增长和利润率。
    • 判断 C：还不能判断。
      • 等 2026 H1 分 IP、库存和同店数据。

  • Pop Mart 的 normalized margin 应该用哪档？

    • 平台型：30%+。
    • 优质消费品：20%-25%。
    • 爆款周期股：10%-15%。

  • 海外扩张是复制能力还是追高？

    • 支持复制能力的证据：成熟店同店、复购、区域利润率。
    • 支持追高的证据：新店多、同店弱、库存高、折扣上升。

3.4 给用户的下一步提示

现在最重要的一条线是：2026 年以后，Pop Mart 能不能证明增长已经从 Labubu 热度迁移到全球 IP 平台能力。

  • 第一证据：分 IP sell-through。
  • 第二证据：库存和折扣。
  • 第三证据：海外成熟店 cohort。
  • 第四证据：非 THE MONSTERS 新品成功率。

只要这四件事没回答，任何估值都只是在把 2025 往未来拖。
